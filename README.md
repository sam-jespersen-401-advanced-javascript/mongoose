#### Running the app

* `npm start`
    * runs `index.js`
* `npm run lint`
    *   runs `eslint`
* `npm test`
    * runs `jest`
* `npm run test-watch`
    *   runs `jest` with the flags `--watchAll --verbose --coverage`

### Setup
##### .env
MONGODB_URI=mongodb://localhost:27017/`db-name`
  
#### Tests
* Unit tests: `npm test`
* Lint tests: `npm run lint`
